# Othello_sfen_viewer
- Othello_sfen_viewerはUSI-Xプロトコル(オセロ版)のsfen文字列を、人間が見やすいように絵で表示するコードです。
- くそコードアレルギーの人は見ないでください。
- USI-Xプロトコル(オセロ版)の詳細はここを参照のこと: https://github.com/YuaHyodo/USI-X-protocol_othello_version
